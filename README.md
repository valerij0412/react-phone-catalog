# Phone Catalog 📱

This is a responsive web application for browsing a catalog of mobile phones. Users can view product details, compare features, and navigate through the catalog effortlessly.

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/valerij0412/react-phone-catalog.git](https://github.com/valerij0412/react-phone-catalog.git)
   cd react-phone-catalog

2. npm install
3. npm start

Features:   
Responsive layout (Mobile, Tablet, Desktop)
Dynamic product rendering
